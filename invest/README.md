## レポート

[`docs/hayashi-new-high-report.md`](./docs/hayashi-new-high-report.md) — 林則行氏の経歴検証、手法（新高値の公式・業績の公式・売りの公式）の内容確認、ジム・ロジャーズ／ラリー・ウィリアムズとの関係検証、暴落サイン・金投資の主張整理、そしてユーザーが保有していたAI生成要約との照合表を含む。各主張には確度ラベル（[A]一次情報／[B]複数二次情報一致／[C]単一ソース・未確認／未確認）を付している。

出典の詳細は [`docs/sources.md`](./docs/sources.md) を参照。

**調査上の重要な制約**: 本調査はWebFetchによる本文取得がプロキシ制限で全滅したため、検索結果のスニペット・要約情報が中心であり、書籍原文の逐語確認は行っていない。詳細はレポート第1章を参照のこと。

## ツールの使い方（`tools/stock-checklist/`）

新高値・業績条件をフォーム入力し、レポートで確認できた条件に基づく機械的な判定結果を表示する個人学習用ツール。ビルド不要、ブラウザのみで完結する。

1. `tools/stock-checklist/index.html` をブラウザにドラッグ＆ドロップで開く（またはローカルサーバ経由でも可）。
2. 銘柄情報・株価チャートの状況・決算短信の数字・企業の質・テーマ性を入力する。
3. 「判定する」を押すと、入力内容に基づく判定結果が表示される。
4. 「この銘柄を保存」でブラウザの `localStorage` に記録し、画面下部の一覧で振り返ることができる。

判定に使われる条件（新高値=過去2年、経常利益成長率20%以上、等）は、レポートで確度[B]以上と評価した項目を中心に設計しているが、林氏本人の著書での一次確認が取れていない項目も含む。判定結果はあくまで学習用の目安であり、実際の投資判断の根拠にしないこと。
