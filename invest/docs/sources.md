# 出典一覧 — 林則行氏「新高値投資」手法 精査レポート

本ファイルは `hayashi-new-high-report.md` の根拠として参照したURL・情報源を、種類別・確度タグ付きで整理したものである。

**確度タグの意味**（レポート本体と共通）:
- **[A]** 一次情報（出版社公式ページの書誌情報、本人・関係者の直接発信で内容が一意に確認できるもの）
- **[B]** 複数の独立した二次情報が一致（ただし多くは本人・出版社発信に遡る可能性がある）
- **[C]** 単一ソースのみ・未確認情報を含む
- **未確認** 検索で該当情報に到達できなかった項目

**共通の制約**: 本調査はWebFetchによる本文取得がプロキシ制限により全滅したため、WebSearchの検索結果スニペット・要約に基づく。書籍原文・一次資料の逐語確認は行っていない。
