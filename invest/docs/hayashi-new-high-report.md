### 1.2 検証方法とその限界（重要）

本調査は次の手順で行った。

1. 林則行氏の著書・経歴・手法・関連人物（ジム・ロジャーズ、ラリー・ウィリアムズ）についてWeb検索を実施し、書店サイト、出版社公式ページ、レビューサイト、投資メディア記事、SNS・ブログ等のスニペット情報を収集した。
2. 収集した情報を突き合わせ、複数の独立したソースで内容が一致するか、単一ソースのみの主張かを区別した。
3. **最大の制約として、WebFetchによる各URLへの本文アクセスがプロキシ制限により全面的に失敗した（実質的に成功率0%）。** そのため本調査は検索結果に表示される見出し・要約・レビュー抜粋（スニペット）のみに基づいており、書籍原文・記事本文を通しで読んだ上での逐語確認は一切行えていない。

この制約の実務的な意味は以下の通りである。

- 「◯◯というルールがある」という記述の多くは、書店のレビュー欄や要約ブログが伝える内容に基づいており、著者本人の原文表現・文脈・例外条件までは確認できていない。
- 数値（%、倍率、期間等）は複数ソースで一致すれば信頼度を上げているが、それでも「関係者・レビュアーが要約した数値」である可能性が残る。
- 一次資料（書籍現物、著者本人の一次発信）への到達を試みたが、独立した第三者による検証記事・監査記録には行き着かなかった項目が多い。特に経歴・実績面でこの傾向が顕著である（第2章参照）。

以上より、本レポートの結論は「現時点のWeb公開情報から合理的に推定できる範囲」にとどまり、確定的な事実認定ではないことを強く留意されたい。

## 3. 手法の全体像

### 3.1 主著『株の公式』の構成

『伝説のファンドマネージャーが教える株の公式 大化け株を見抜く13のルール』（ダイヤモンド社、2010年9月、ISBN 9784478014509）の章構成は複数の書誌情報で一致しており確度は高い[B]。

| 章 | タイトル |
|---|---|
| 第1章 | 公式に代入するだけで株で勝てる |
| 第2章 | 新高値をつけた銘柄しか買ってはいけない！ |
| 第3章 | 3つの業績を見れば、スター株が見つかる |
| 第4章 | 絶対に守らなければいけない売りの公式 |
| 付録1 | チェックリストで準備万全 |
| 付録2 | 必要なデータはどこで手に入れるのか |
| 付録3 | 売りどきを知るための情報源 |

**「13のルール」の内訳は一次資料に到達できず未確認である。** 買いの公式が「8つ」とする情報源と「9つ」とする情報源が併存しており、これに売りの公式（複数ソースで「4つ」と一貫）を足しても13にきれいに一致するかは、検索の範囲では確認できなかった。ソースの揺れは「新高値であること」と「シグナル出現位置・6割戻り条件」を1項目として数えるかどうかの差である可能性があるが、これも推定にとどまる。

### 3.2 手法の骨格

林氏の手法は、大きく分けて次の3要素から成る。

1. **新高値の公式**（第2章）: 株価が新高値を更新した銘柄のみを投資対象とする。
2. **業績の公式**（第3章）: 決算短信の数字（売上高・利益）から急成長企業を発見する。
3. **売りの公式**（第4章）: テクニカルサインと業績悪化を基準に機械的に売却する。

続編『伝説のファンドマネージャーが実践する 株の絶対法則』（ダイヤモンド社、2012年頃、ISBN 9784478022733）では、この骨格をベースにテクニカル分析・大局観・資産防衛の要素が追加され、ファンダメンタルズ指標が簡素化されたとの言及がある（数字は「売上高10%以上・経常利益20%以上」に整理されたとする情報あり）[B]。ただし「4指標から2指標に簡素化された」という経緯自体は単一ソース由来であり確度は限定的[C]。

『株の絶対法則』の章構成（複数ソースで一致、確度B）は以下の通り。

| 章 | タイトル（推定含む） |
|---|---|
| 第1章 | 大化け株は一目でわかる |
| 第2章 | 買いの核心はビッグチェンジ |
| 第3章 | 買ってはいけない銘柄 |
| 第4章 | テクニカル分析はこれだけに絞る |
| 第5章 | 売りの小技で成績アップ |
| 第6章 | 大きな流れをつかめば乱高下相場も怖くない |
| 第7章 | 「資産を守る」ことこそ最大の攻め＋付録チェックリスト |

## 4. 買いのルール（詳細）

### 4.1 新高値の定義

| 項目 | 内容 | 確度 |
|---|---|---|
| 新高値の期間 | **過去2年**の高値を更新した時点 | B |
| 上場来高値限定か | 「上場来高値」を要求する記述は確認できず。過去2年基準が中心と見られる | B（否定側の確認） |
| シグナル出現位置 | 過去最高値からの戻りが**6割以上**必要という条件の言及あり | C（単一ソース） |

### 4.2 業績条件（決算短信の数字を使う部分）

ユーザー提供要約にある「決算短信の4つの数字を転記するだけ」という主張について、検索で確認できた内容は次の通りである。

- 基本構造として、決算短信のサマリーから「売上高」と「利益（複数種）」を転記し、複雑な企業価値計算をせずに高成長株を発見する、という位置づけは複数ソースで一致する[B]。
- 「4つの数字」の内訳が売上高・営業利益・経常利益・当期純利益の4項目を指すという直接の明言には到達できず、**決算短信の標準的な構成から推定される内容**にとどまる[B（推定）]。
- 数値の閾値としては次が複数ソースで確認できる。

| 指標 | 条件（閾値） | 期間 | 確度 |
|---|---|---|---|
| 経常利益（成長性） | 過去5〜10年間で年平均7%以上の成長 | 長期 | B |
| 経常利益（安定性） | 過去5〜10年間で減益がない、またはごく少ない | 長期 | B |
| 経常利益（短期） | 前年同期比20%以上の伸び | 直近1〜2年 | B |
| 売上高 | 前年同期比10%以上の伸び | 直近2〜3四半期 | B |
| 利益（四半期） | 前年同期比20%以上の伸び | 直近2〜3四半期 | B |
| PER | 60倍以下 | — | B |

- 「何が起きても成長を続ける銘柄を選ぶ（不況・円高でも増益）」という定性的な条件も見られるが単一ソースであり確度は限定的[C]。
- 4項目それぞれを厳密に切り分けた個別の閾値、および四半期単位での経常利益成長率の細かな条件は未確認である。

### 4.3 エントリーのタイミング

- **新高値更新の当日には買わず、翌営業日の始値の0.5〜1.0%下に指値を置く**、という具体的なエントリー手法が続編『株の絶対法則』の内容として紹介されている[B〜C]。この情報は要約ブログ経由が中心であり、複数の独立した一次情報での裏付けはやや弱い。
- もみ合い（ベース）期間の長さや、ブレイク時の出来高倍率についての定量的な条件（例: 平均出来高の何倍以上、など）は未確認である。オニールのCAN-SLIM等、類似の新高値ブレイクアウト手法では出来高急増を重視する傾向があるが、林氏の著書での明示的な数値条件は確認できなかった。

## 5. 売りのルール

| ルール | 内容 | 確度 |
|---|---|---|
| 基本方針 | 売りは原則としてテクニカルの売りサインで行う | B |
| 業績悪化 | 直近の利益成長率（前年同期比）が20%を下回ったら売り | B |
| 不正・事故 | 会社の不正が判明したら即売り、事故発生時も原則売り | C（単一ソース） |
| 損切り | 買値から**8%前後**下落した時点で機械的に損切り | B〜C（下記参照） |

### 5.1 損切り基準における情報源の食い違い（要注意）

損切り基準の数値については、調査①（13のルール系統）と調査②（4つの数字系統）で評価が割れた。両論を正直に併記する。

- 調査①では、複数の独立したレビュー・書評サイトで「損切りは買値から8%前後」という数値が一致しており、確度[B]と評価した。
- 調査②では、林氏本人の著書に明記された損切り%の直接記述には到達できず、確認できたのは**実践者経由**の数値であった。具体的には、ダイヤモンド・オンラインの実践者連載「理系男子の株式投資」で紹介される「買値から8%下落」という数値と、別の実践者（kenmo氏、林氏＋オニール手法を実践するとされる人物）による「①買値から10%下落 ②新高値後のボックス圏の底割れ」という数値の2系統が存在し、いずれも確度[C]と評価した。

**結論として、「8%」という数値は複数箇所で登場し一定の収斂は見られるものの、林氏本人の著書における一次記述としての確認は取れていない。** 「損切りが最重要である」という定性的な方針自体は複数ソースで一致しているが、具体的なパーセンテージを機械的に適用する前に、この数値が実践者による解釈・要約である可能性を踏まえておくべきである。

## 9. 個人投資への実践適用ガイド

以下は、本調査で確度[B]以上とされた要素を中心に、個人投資家が実務でどう活用しうるかの一案である。**手法固有の数値（損切り8%、新高値=過去2年、等）は本人著書での一次確認が取れていない項目を含むため、実際に使う際は自己判断で慎重に検証すること。**

### 9.1 週次ルーチンの例

1. **スクリーニング**: 証券会社のスクリーナー機能で「過去2年の高値を更新した銘柄」を抽出する（多くの国内証券会社のスクリーナーで「52週高値」等に類する条件設定が可能。「過去2年」の条件が直接設定できない場合は「上場来高値」や「52週高値更新」で代用しつつ、チャートで過去2年のレンジを目視確認する）。
2. **業績確認**: 抽出された銘柄について、直近の決算短信を開き、売上高の前年同期比伸び率（目安10%以上）と経常利益の前年同期比伸び率（目安20%以上）を確認する。可能であれば過去5〜10年の経常利益推移で成長性・安定性（大きな減益がないか）も確認する。
3. **チェックリストでの判定**: 本リポジトリの `invest/tools/stock-checklist/` にあるツールに、新高値の有無・業績数値・出来高等を入力し、機械的な判定結果を得る。ツールはあくまで公開情報を基にした学習用の補助であり、判定結果を鵜呑みにしない。
4. **エントリー**: 買うと判断した場合、新高値更新の当日ではなく翌営業日の寄り付き付近を意識し、指値は始値からやや下（0.5〜1.0%程度）に置くという考え方がある（確度B〜C、上述の通り一次確認は弱い）。
5. **損切りラインの事前設定**: エントリー前に、買値から一定率（8%前後という情報が複数あるが本人著書での一次確認はなし）を下回った場合の損切りルールを機械的に決めておく。感情的な判断を排除するため、可能であれば証券会社の逆指値注文機能を使う。
6. **保有中の売りサイン監視**: テクニカルな売りサイン（トレンド転換、移動平均線割れ等の一般的なチャート分析）と、四半期ごとの利益成長率が20%を下回っていないかを継続的に確認する。

### 9.2 資金管理の一般原則

林氏の手法固有の資金管理ルール（分散銘柄数、1銘柄あたりの投入比率、保有期間の目安等）は公開情報からは確認できなかった（第10章参照）。そのため、以下は一般的なリスク管理の原則として付言するにとどめる。

- 1銘柄への投入比率を過度に高くしない（一般に個人投資家では1銘柄あたり資産の10〜20%程度を上限とする考え方が広く紹介されるが、これは林氏固有の主張ではなく一般的なリスク管理の目安である）。
- 損切りルールを事前に機械的に定め、感情的な判断（「もう少し待てば戻るはず」等）で先送りしない。
- 新高値ブレイクは「だまし」（ブレイクの失敗、いわゆるフェイクアウト）が一定確率で発生する手法であることを前提に、1回の損失で致命的な打撃を受けない資金配分にする。

### 9.3 チェックリストツールの活用

本リポジトリの `invest/tools/stock-checklist/index.html` は、上記の新高値・業績条件をフォーム入力し、機械的な判定結果を表示する個人学習用ツールである。使い方は `invest/README.md` を参照のこと。ツール自体が投資助言ではなく、あくまで本調査で確認できた公開情報の範囲内でのチェックリスト化にとどまる点に留意する。

### 9.4 適用にあたっての心構え

- 本手法は個別成長株への集中投資に近い性格を持ち、インデックス投資に比べてボラティリティ・銘柄選定リスクが高い。資産全体のごく一部（いわゆるサテライト部分）での実践にとどめ、生活防衛資金や中核資産（インデックスファンド等）とは明確に分離することが望ましい。

## 10. 手法の限界と批判的検証

### 10.1 生存者バイアスと自己申告実績

- 著書で紹介される成功事例（10倍株の発掘等）は、後知恵で選ばれた成功例である可能性を排除できない。失敗事例・損切りに終わった銘柄がどの程度の頻度で発生していたかについての情報は確認できなかった。
- 経歴・運用実績が自己申告に依存しており、独立した監査記録が存在しない点は前述の通り（第2章）。手法の有効性を判断する上で、著者自身の実績を強い根拠として採用することには慎重であるべきである。

### 10.2 高値掴みリスク（だましブレイク）

新高値ブレイクアウト系の手法全般に共通する弱点として、価格が新高値を更新した直後に反落する「だましブレイク（フェイクアウト）」が一定頻度で発生することが知られている。この場合、新高値を確認してから買う手法は構造的に「高値で買って損切りする」パターンを一定割合含むことになり、損切りルールの厳格な運用が手法全体の収益性を左右する。林氏の著書で紹介される「もみ合い（ベース）期間の長さ」「出来高急増」等の付随条件がだましブレイクをどの程度回避できるかについて、定量的な検証データは確認できなかった。

### 10.3 レンジ相場での不振

新高値ブレイクアウト系の手法（ラリー・ウィリアムズのボラティリティ・ブレイクアウト手法を含む）は、一般にトレンドの出やすい相場・高ボラティリティ環境で機能しやすく、方向感のないレンジ相場では機能しにくいとされる[B]。日本株市場が長期のレンジ相場・低ボラティリティ局面に入った場合、この手法の有効性が低下する可能性は理論的に想定される。

### 10.4 ウィリアム・オニールのCAN-SLIMとの類似性

林氏の手法は、米国の投資家ウィリアム・オニールが提唱するCAN-SLIM投資法と構造上の類似点が多い。

| 要素 | 林氏の手法 | CAN-SLIM |
|---|---|---|
| 株価の基準 | 新高値更新（過去2年） | 新高値更新（52週） |
| 利益成長 | 経常利益前年比20%以上 | 直近四半期利益の大幅増（一般に25%以上とされる） |
| 売上成長 | 売上高前年比10%以上 | 売上高の加速的成長 |
| エントリー | 翌日寄り付き付近、指値でのブレイクアウト買い | ピボットポイント（ベースからのブレイクアウト）での買い |
| 損切り | 買値から一定率（複数の情報で8%前後） | 一般に買値から7〜8% |

実際、本調査で参照した実践者記事（jbpress.ismedia.jp/articles/-/88604）の著者は「林氏＋オニールが教科書」と明言しており、両者を組み合わせて実践している投資家も存在する。この類似性自体は、手法の妥当性を否定するものではなく、むしろ「新高値・業績加速・出来高」という組み合わせが複数の実務家によって独立に有効とされてきた古典的な成長株投資アプローチであることを示唆するとも解釈できる。一方で、林氏独自の付加価値（何がオニール由来と異なるのか）については、公開情報から明確に切り分けることは難しい。

### 10.5 2026年時点の市場環境での留意点

- 決算短信の様式は東京証券取引所・金融庁の方針変更により時期によって変わりうる。本手法が想定する「決算短信サマリーの数字」の位置・呼称が、著書執筆時点（2010〜2013年頃）と現在とで異なる可能性があるため、実際の適用時には最新の様式を確認する必要がある。
- 2022年の東証市場区分再編（プライム・スタンダード・グロース市場への移行）により、旧・東証一部を前提とした銘柄選定の母集団や指数の考え方が変化している。「日経平均」「TOPIX」等の指数比較を伴う暴落サイン分析を行う場合、この点を踏まえる必要がある。
- 近年のインデックス投資・NISA拡充等、個人投資家を取り巻く制度環境が2010年代前半の著書執筆時点から大きく変化しており、個別銘柄の新高値投資という高リスク・高リターンの戦略を選択する際は、資産全体におけるこの手法の位置づけ（コア・サテライト戦略でいうサテライト部分に限定する等）を検討することが望ましい。

### 10.6 有料講座への勧誘導線と情報の非対称性

- 現在の「投資部」の集客導線は、無料コンテンツ（YouTube等）→ 有料コース（275,000円）という構成である。無料コンテンツでは暴落予測等の不安喚起的なメッセージが多く見られる一方、手法の核心部分（空売り・底値買いの具体的手法等）は非公開・有料領域に置かれている[B]。
- このため、本レポートで検証できた範囲は「公開されている書籍・記事の情報」に限られ、現在の有料講座で実際にどのような内容が教えられているかは検証の対象外である。講座の価値判断は本レポートの範囲外とする。
- 一般論として、投資教育サービスを検討する際は、(1) 運営会社の実在性・実績、(2) 講師の実績の第三者検証可能性、(3) 過去受講者の第三者評価、(4) 返金・解約条件、を契約前に確認することが望ましい。
