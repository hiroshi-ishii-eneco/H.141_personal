# 漢字練習プリント生成ツール

## プロジェクト概要

小学生・中学生向けの漢字練習プリントを A4 縦・1 ページ 20 問 (2 列 × 10 行) で
ブラウザから印刷できるツールです。バニラ JavaScript / HTML / CSS のみで動作し、
ビルドステップや外部依存はありません。学年・出題範囲・モード (読み / 書き)・
解答欄スタイル・十字リーダーなどを切り替えて、家庭学習用に好みのプリントを
すぐに用意できます。

## 使い方

### 開き方

- 一番手軽な方法: `index.html` をブラウザにドラッグ & ドロップで開く。
- ローカルサーバで開く場合 (フォントや一部ブラウザの制約回避に推奨):

  ```sh
  cd path/to/this/repo
  python3 -m http.server 8000
  # ブラウザで http://localhost:8000/ を開く
  ```

### 操作

1. 画面上部の操作パネルで学年・出題範囲・モード・解答欄スタイルなどを選ぶ。
2. 「問題を新しく作る」を押すたびに出題がシャッフルされる。
3. 採点用の解答が必要なら「解答ページを含める」を ON にする
   (出題ページ群の末尾に解答ページが 1〜2 枚追加される)。
4. 「印刷する」を押して印刷ダイアログから出力する。

### 印刷時の注意

- ブラウザの印刷ダイアログで「ヘッダ / フッタ」のチェックを必ず外す
  (URL や日付がページ余白に印字されるのを防ぐ)。
- 「背景のグラフィック」は ON / OFF どちらでも構わない (本ツールは背景塗りに依存しない)。
- 用紙サイズは A4、向きは「縦」、余白は「既定」または「12mm」相当に。

### 設定永続化

- 学年やモードなどの設定は `localStorage` のキー
  `kanji_practice_settings_v1` に自動保存されます。
- 次回起動時には前回の設定が復元されます。
- 設定をリセットしたいときはブラウザの開発者ツール →
  Application → Local Storage から該当キーを削除してください。

## データの拡張方法

各学年の単語データは `data/kanji-g1.js` 〜 `data/kanji-jhs3.js` に
それぞれ 1 ファイルずつ用意されています。新しい単語を追加するには対象学年の
ファイルを開き、`window.KANJI_DATA_GN` 配列に以下の形でエントリを追記します。

```js
{
  kanji: "学",
  readings: { on: ["ガク"], kun: ["まな(ぶ)"] },
  words: [
    {
      word: "学校",
      reading: "がっこう",
      sentence: "あたらしい○○にかよう。",
      printable: true
    }
    // ...
  ]
}
```

- `sentence` 中の `○` (1 つ以上連続) が空欄プレースホルダ。アプリ側で
  正答文字数に合わせて自動的に長さ調整 (書きモード) または漢字に
  置換 (読みモード) されます。
- 正答が 5 文字を超えるもの、または `printable: false` のものは出題から
  自動除外されます。

配当漢字定義 (`data/grade-assignments.js`) は将来 MeCab / Sudachi 連携の
正規ソースとして使う想定で、現状は空配列のまま運用しています。

## データ品質検証

`scripts/validate-kanji-data.mjs` で各学年データの整合性を機械チェックできます。
Node.js 18+ が必要です。

```sh
node scripts/validate-kanji-data.mjs           # 全学年検証
node scripts/validate-kanji-data.mjs g3        # 学年指定
node scripts/validate-kanji-data.mjs --verbose # 検出内容の詳細表示
node scripts/validate-kanji-data.mjs --strict  # warning も exit code 2 で失敗扱い
node scripts/validate-kanji-data.mjs --json    # JSON 出力
```

検証項目:
- word に対象漢字が含まれているか
- sentence の `○` プレースホルダが 1 箇所だけ存在するか
- reading がひらがな (+ 長音符) のみで構成されているか
- word が `MAX_ANSWER_LEN` (5 字) を超えないか
- sentence が学年別の推奨上限 (p95+1 で校正) に収まるか

最新の検証結果: errors 0 / warnings 49 / **clean rate 99.20%** (95% 目標達成済)。

## 既知の制限

- 中学 1 年〜中学 3 年のデータは未整備で、対応ファイルは空配列のままです。
  該当学年を選ぶと「この学年のデータは準備中です。」と表示されます。
- 採点機能はありません (各問右上に自己採点用の 6mm 角の小枠を印字します)。
- 出題は端末内の擬似乱数によるシャッフルで、再現性のあるシード指定は未対応。
- 同一単語が 1 プリント内に重複出題される可能性があります
  (現状はフラットなランダム抽出のため)。
- 印刷フォントは OS / ブラウザに依存します。下記の推奨フォントを
  インストールするとレイアウトが安定します。
